# Cognitive Design

O Cognitive Design é um site voltado a explorar e demonstrar conceitos de design cognitivo aplicados à interface e experiência do usuário. O objetivo é apresentar padrões, componentes e interações que facilitam a compreensão, reduzem a carga mental e promovem decisões mais rápidas e acessíveis, unindo princípios de psicologia cognitiva a um design de UI/UX moderno.
